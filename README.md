# Network-Security-Assessment
Automated network reconnaissance and security assessment tool built in Python.

What it does:
→ Host discovery across CIDR ranges
→ Open port enumeration and service fingerprinting
→ Basic vulnerability flagging on identified services
→ Outputs structured assessment reports ready for review

Built to automate the manual recon phase of a penetration test — reducing time spent on initial enumeration and producing consistent, structured output across assessments.
